# measurement-converter

Functions to convert kg to li and li to kg.

## Installation

```
npm install measurement-converter
```

## Usage

```
import { kgToLi, liToKg } from 'measurement-converter'

kgToLi(10); // 4.5360
liToKg(10); // 22.0460
```
